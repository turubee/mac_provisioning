# mac provisioning playbook

```bash
$ xcode-select --install
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew doctor
$ brew update
$ brew install python
$ brew install ansible
$ git clone https://github.com/turubee/mac_provisioning
$ cd mac_provisioning
$ bash provision.sh
```
